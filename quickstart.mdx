## Run your first scan

<Steps>
  <Step title="Open a terminal in the project folder">
    <Tabs>
      <Tab title="Linux / macOS">
        ```bash
        cd BeCode-Python_Project
        ```
      </Tab>
      <Tab title="Windows">
        ```bash
        cd BeCode-Python_Project
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Run the simplest possible scan">
    Pass a target hostname or IP address. Without any extra flags, the scanner probes ports 1–1024 using 100 concurrent threads at a rate of 10 probes per second:

    ```bash
    python Port_scanner.py scanme.nmap.org
    ```

    <Note>
      On Linux and macOS, you may need to use `python3` instead of `python` depending on your system configuration.
    </Note>
  </Step>

  <Step title="Watch the terminal output">
    As ports are probed, results print in real time:

    ```
    Port 22   → ssh is open
    Port 25   → smtp is closed
    Port 80   → http is open
    Port 135  → Service name NOT FOUND is closed
    Port 443  → https is open
    Port 631  → ipp is filtered (timeout)
    ...
    Results saved to scan_results_2024-01-15_10-30-00.json

    Port Scanning is Complete.
    ```

    Each line shows:
    - The **port number**
    - The **service name** resolved from your OS's service database (or `Service name NOT FOUND` for unregistered ports)
    - The **status**: `open`, `closed`, or `filtered (timeout)`
  </Step>

  <Step title="Find your output file">
    When no output filename is specified, results are saved automatically to a timestamped JSON file in the current directory:

    ```
    scan_results_2024-01-15_10-30-00.json
    ```

    Open it to see the full structured results:

    ```json
    {
        "target": "scanme.nmap.org",
        "scan_results": [
            {"port": 22, "status": "open"},
            {"port": 25, "status": "closed"},
            {"port": 80, "status": "open"},
            {"port": 443, "status": "open"},
            {"port": 631, "status": "filtered"}
        ]
    }
    ```

    Results are sorted by port number in ascending order.
  </Step>

  <Step title="Run a targeted scan with a named output file">
    To scan only ports 80 and 443 and save results as a plain-text file:

    ```bash
    python Port_scanner.py scanme.nmap.org -s 80 -e 443 -o web_scan.txt
    ```

    The `-s` and `-e` flags set the start and end of the port range. The `-o` flag sets the output filename — the extension (`.txt`, `.json`, or `.csv`) determines the format.

    The resulting `web_scan.txt` file looks like this:

    ```
    PORT SCAN RESULTS
    =================
    Target : scanme.nmap.org
    Date   : 2024-01-15_10-30-00
    =================

    Port 80     | http                 | open
    Port 443    | https                | open
    ```
  </Step>
</Steps>
